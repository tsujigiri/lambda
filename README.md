# Lambda

Lets you write `λ` for `lambda`. That is all.

## Usage

```ruby
my_lambda = λ {|foo| foo.reverse }
my_lambda["bar"] # => "rab"
```

## Installation

Add this line to your application's Gemfile:

    gem 'lambda'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install lambda
